# ClothingStore
Dự án quản lý bán hàng quần áo - được phát triển bởi Hoang.

## Giới thiệu
Ứng dụng giúp quản lý kho hàng, đơn hàng, và khách hàng trong cửa hàng thời trang.

## Cách sử dụng
Chạy chương trình bằng cách biên dịch và chạy file `main.cpp`.
